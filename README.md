# conversations-python
Python version of my conversations library.

## Usage

### Install 

```bash
pip install git+ssh://git@github.com/Kai-Karren/conversations-python.git
```

## Install from Source

### Requirements

```bash
pip install build
```

### Build

```bash
python -m build
```

### Install

```bash
pip install --force dist/conversations-0.0.4-py3-none-any.whl
```
